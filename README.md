## 🚀 如何运行我的代码

1. 确保已安装 Python 3.10 或更高版本  
2. 克隆本仓库：  
   `git clone https://github.com/good3s/python-learning.git`  
3. 进入对应文件夹，运行：  
   `python 文件名.py`
